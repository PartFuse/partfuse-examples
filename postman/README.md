# PartFuse Postman Collection

## Usage

1. **Import** `PartFuse.postman_collection.json` into Postman.
2. **Import** `PartFuse.postman_environment.json` into Postman.
3. Select the "PartFuse Environment" from the top-right dropdown.
4. Edit the environment variables: set `rapidapi_key` to your actual API key.
